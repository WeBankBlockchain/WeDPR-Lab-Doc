## SCD示例快速体验

完成[安装步骤](./installation.md)之后，进入示例目录，运行选择性认证披露Demo。

 ```bash
cd WeDPR-Lab-Core/solution/selective_certificate_disclosure
cargo run
 ```

cargo run之后按照demo指引，设置中文或英文作为演示语言，按步骤输入即可进行体验。
